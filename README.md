# Auto-Dark-Light
# Windows 深色模式自动切换脚本 (PowerShell)

## 概述
本脚本通过调用公开的日出/日落时间 API，自动计算当前所在地的日照状态，并在**日落时**将 Windows 系统主题切换为**深色模式**，在**日出时**切换回**浅色模式**。全程无需人工干预，适合放在任务计划程序(taskschd.msc)中定时执行。

## 适用场景
- 护眼与舒适视觉：根据环境光线自动适配界面色调。
- 自动化办公/娱乐：避免手动切换主题的繁琐操作。
- 与动态壁纸、夜间模式联动，打造全时段自适应桌面。

## 工作原理
1. **获取位置**：获取用户当前经纬度与时区。
2. **计算日照**：调用 `sunrise-sunset.org` API，根据经纬度获取当日日出/日落时间（UTC）。
3. **转换本地时间**：将 UTC 时间转换为本地时区时间。
4. **判断当前时刻**：比较当前时间与日出/日落时间。
   - 当前时间 > 日落 或 当前时间 < 日出 → **深色模式**
   - 其他时段 → **浅色模式**
5. **应用主题**：通过修改 Windows 注册表项，切换系统与应用（UWP）的主题颜色。

## 使用前提
- Windows 10 / Windows 11（已测试 22H2+）
- PowerShell 5.1 或更高版本（建议 7.x）
- 需要**联网**以获取位置与日出日落数据
- 无需管理员权限（但切换主题可能需要重启资源管理器生效，建议以当前用户身份运行）
